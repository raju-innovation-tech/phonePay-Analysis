# phonePay-Analysis
phonePay Analysis
📌 Project Overview

The PhonePe Transaction Analysis Dashboard is an interactive Power BI dashboard designed to analyze digital payment transactions across different PhonePe services. It provides insights into transaction amount, successful transactions, failed transactions, service-wise performance, monthly trends, and failure reasons.

This dashboard helps stakeholders monitor transaction performance, identify high-performing services, and understand the major causes of failed transactions.

Service-wise Analysis
Loans contribute the highest transaction amount.
Insurance is the second most used service.
Recharge & Bills have comparatively lower transaction value.
Monthly Trend
Transaction amount peaks during July.
November records the lowest transaction volume.
